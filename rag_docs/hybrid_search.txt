# 문서 8: 하이브리드 검색 전략

하이브리드 검색은 키워드 기반 검색과 벡터 검색을 결합하는 기법입니다. BM25 같은 키워드 검색으로 후보를 추린 후, 벡터 검색으로 재순위를 매깁니다.

turbovec의 allowlist 기능은 하이브리드 검색을 효율적으로 지원합니다. 키워드 검색 결과를 allowlist로 전달하면 SIMD 커널이 직접 필터링합니다.

32-벡터 블록 단위로 early-exit가 동작하여 허용된 슬롯이 없는 블록은 건너뜁니다. 이는 선택적 필터에서 대부분의 SIMD 비용을 절약합니다.

BLOCKS_SKIPPED_BY_MASK 카운터로 스킵 통계를 확인할 수 있어 하이브리드 검색의 효율성을 모니터링할 수 있습니다.
