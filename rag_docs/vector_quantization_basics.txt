# 문서 5: 벡터 양자화 기초

벡터 양자화는 고차원 벡터를 저차원 표현으로 압축하는 기술입니다. 이는 메모리 사용량을 줄이면서도 검색 정확도를 유지하는 데 필수적입니다.

양자화의 핵심은 각 좌표값을 소수의 비트로 표현하는 것입니다. 2비트는 4단계, 3비트는 8단계, 4비트는 16단계로 값을 표현합니다. 비트 수가 적을수록 압축률은 높지만 왜곡이 커집니다.

TurboQuant는 데이터 무관한 양자화기를 사용합니다. 이는 훈련 데이터를 보지 않고도 양자화할 수 있다는 의미입니다. 회전 행렬을 적용한 후 각 좌표가 알려진 Beta 분포를 따르도록 만들어 이론적으로 근사 최적의 왜곡을 달성합니다.
